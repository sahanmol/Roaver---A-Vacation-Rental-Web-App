# Roaver - A Vacation Rental Web App

Roaver is a modern and user-friendly vacation rental platform that allows users to explore, list, and book unique stays around the world.  
Built with a clean interface and secure backend architecture, Roaver provides a seamless experience for both travelers and property owners.

---

## ✨ Features

- 🔐 User Authentication & Authorization
- 🏠 Add, Edit, and Delete Property Listings
- 📸 Image Upload Support
- ⭐ Reviews & Ratings System
- 📱 Fully Responsive Design
- 🛡️ Secure Backend & Database Integration
- 🔎 Easy Property Browsing Experience

---

Roaver - A Vacation Rental Web App

Roaver is a modern vacation rental web application where users can explore, list, and book amazing stays around the world.
The platform provides a smooth and user-friendly experience for travelers and property owners.

Features
User Authentication & Authorization
Add, Edit & Delete Property Listings
Image Upload Support
Reviews & Ratings System
Responsive UI Design
Secure Backend & Database Integration
